# Analisis Data Mining: Credit Card Fraud Detection

Project ini berisi analisis data mining menggunakan dataset Credit Card Fraud Detection.

## Dataset
Dataset yang digunakan dalam analisis ini adalah **Credit Card Fraud Detection** yang dapat diunduh melalui tautan berikut:
[https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data)

Karena ukuran dataset (creditcard.csv) melebihi batas ukuran file GitHub (100MB), file tersebut tidak disertakan dalam repositori ini. Silakan unduh dataset dari tautan Kaggle di atas dan simpan di dalam folder `datasets/` dengan nama `creditcard.csv` jika Anda ingin menjalankan kode pada repositori ini.
